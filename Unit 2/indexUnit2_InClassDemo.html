Paragraph Elements

The p tag used for general purpose text on our pages. This tag is also concidered an inline element. Meaning by default other elements can be placed adjacent to it.



Blockquote Element

The Blockquote is used to define a paragraph when it is desired to have the the paragraph indented

Blockqutes have roughly a 40 pixel indent and are great for quoting areas of text in a passage. Blockquotes are block elements and therefore by default will have a blank line directly before and after the element is used.


Line Breaks

You may not have noticed but line breaks have been used throughout this entire document as a menas to help organize the content and make it easier to read.

Line Breaks do not have a close tag and therefore are known as an empty element.

		HTML Links - The target Attribute

			The target attribute specifies where to open the linked document.

	The target attribute can have one of the following values:

    _blank - Opens the linked document in a new window or tab
    _self - Opens the linked document in the same window/tab as it was clicked (this is default)
    _parent - Opens the linked document in the parent frame
    _top - Opens the linked document in the full body of the window
    framename - Opens the linked document in a named frame
